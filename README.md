# The Crimson Hunt 🔴

A Vampire Survivors style action game built in Godot Engine.

## About
Hunt endless waves of slimes in a crimson world. Survive as long as possible while managing your health, leveling up, and unleashing the Crimson Domain to wipe everything on screen.

## Original Mechanic — Crimson Domain
Unlock at Level 3. Activate to trigger a 5 second screen shake buildup that grows more violent every second. At the peak — a red flash wipes every enemy on screen instantly.
Inspired by Domain Expansion from Jujutsu Kaisen.

## Built With
- Godot Engine 4
- GDScript
- Custom audio design — signature BGM, ambient gameplay track, layered SFX

## Features
- Dynamic difficulty scaling over time
- Elite slime spawning after 5 minutes
- XP and leveling system with speed boosts
- Recoil, muzzle flash and camera shake game feel
- Original Crimson Domain mechanic
